# Snake Game

A simple Snake game built with Python and Pygame.

## How to Play

1.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```
2.  **Run the game:**
    ```bash
    python snake.py
    ```

## Controls

*   **Arrow Keys:** Move the snake.
*   **C:** Play again after losing.
*   **Q:** Quit the game.
